Price Input Fields - Visual Test

Testing: This page demonstrates the professionally styled price input fields for the bid modal. Click inside each input to see the focus state.

Fixed Price Mode

When user selects "Fixed Price" pricing mode:

Price Per Item
Enter your bid price per item
Blue Border - Primary Price Input

Variable (Premium to Spot) Mode

When user selects "Variable (Premium to Spot)" pricing mode:

Premium Above Spot
Amount to add above current spot price per unit
No Lower Than (Price Floor)
Minimum price floor - your bid won't trigger if spot + premium falls below this amount
Green Border - Premium Input
Orange Border - Floor Price Input

Styling Features

  • $ Prefix: All price inputs show a dollar sign prefix
  • Color Coding:
    • Blue border = Fixed price (static mode)
    • Green border = Premium above spot (variable mode)
    • Orange border = Floor price (variable mode)
  • Large, Bold Text: 16px font, weight 600 for easy readability
  • Focus States: Border color darkens with shadow glow on focus
  • Proper Spacing: 28px left padding for $ symbol + text
  • Professional Appearance: 8px border radius, 2px border width
