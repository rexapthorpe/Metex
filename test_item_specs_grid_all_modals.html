Responsive Testing

To test responsive behavior:

  1. Open browser DevTools (F12)
  2. Enable device toolbar (responsive mode)
  3. Resize viewport to different widths:
    • > 768px: 2 columns (desktop)
    • 481-768px: 2 columns (tablet)
    • ≤ 480px: 1 column (mobile)
  4. Verify labels and values stay on same horizontal line
  5. Confirm last items in each column have no bottom border
